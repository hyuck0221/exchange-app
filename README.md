<h1 align="center">환율 계산기</h1>

<p align="center">
해외여행이나 직구할 때, 원화로 얼마인지 바로 알 수 있는 환율 계산기 앱입니다.
</p>

---

## 이런 분들께 추천해요

- 해외여행 전 현지 화폐와 원화를 빠르게 환산하고 싶은 분
- 해외 직구 시 가격이 한국 돈으로 얼마인지 바로 확인하고 싶은 분
- 환율을 직접 입력해서 계산하고 싶은

## 주요 기능

### 실시간 환율 자동 적용

인터넷에 연결되어 있다면 최신 환율을 자동으로 불러와 계산합니다.
특정 날짜의 환율도 조회할 수 있어요.

### 환율 직접 입력

오프라인 환경이거나 특정 환율로 계산하고 싶을 때, 환율을 직접 입력해서 사용할 수 있어요.

### 양방향 계산

- 외화 → 원화
- 원화 → 외화

어느 쪽이든 입력하면 반대쪽 금액이 바로 계산됩니다.

### 지원 통화

| 통화  | 국가    |
|-----|-------|
| USD | 미국 달러 |
| JPY | 일본 엔  |
| VND | 베트남 동 |
| CNH | 중국 위안 |

> API 연동 모드에서는 더 다양한 통화를 지원합니다.

## 사용 방법

### 처음 앱을 켰을 때

1. **모드 선택**
    - **자동 (API 연동)**: 실시간 환율을 자동으로 가져옵니다
    - **수동 입력**: 원하는 환율을 직접 입력합니다

2. **통화 및 환율 설정**
    - 자동 모드: 원하는 통화와 날짜를 선택하면 환율이 자동 입력됩니다
    - 수동 모드: USD, JPY, VND, CNH 중 하나를 선택하고 환율을 입력합니다

3. **계산 시작!**

### 메인 화면

- 상단 입력칸: 외화 금액 입력
- 하단 입력칸: 원화 금액 입력
- 어느 쪽이든 숫자를 입력하면 반대쪽이 자동 계산됩니다
- 오른쪽 상단 설정 아이콘: 설정 화면으로 이동

### 설정 화면

- 통화 변경
- 환율 직접 수정 (잠금 해제 후)
- API에서 새 환율 불러오기
- 처음부터 다시 설정

<br>

## 특별한 점

- **큰 금액도 쉽게**: 억, 만 단위로 금액을 함께 표시해줘서 큰 숫자도 한눈에 파악할 수 있어요
- **입력값 기억**: 앱을 껐다가 켜도 마지막에 입력했던 금액이 유지됩니다
- **밝은/어두운 테마**: 기기의 다크모드 설정을 자동으로 따라갑니다
